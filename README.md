# Exp - 5 STUDENT ENROLLMENT SYSTEM

# AIM:
To design and develop a system that automates the process of student enrollment, managing student records, course details, and enrollment status efficiently.

# SRS (Procedure):
1. Start the system and log in as admin or staff.
2. Enter new student details such as name, ID, and contact info.
3. Display available courses for enrollment.
4. Select and assign courses to the student.
5. Store enrollment details in the database.
6. Allow students to view or update their information.
7. Generate and display enrollment confirmation report.
8. End the process after successful enrollment.
